What's supported / what's not
  • Protocols handled: VLESS, VMess, Trojan, Shadowsocks, Hysteria2.
  • Not yet: VLESS Reality, TUIC, AnyTLS.
  • Surge / QuantumultX / Loon are partial-coverage outputs — VLESS and Hysteria2 nodes are dropped silently because those clients have no native mapping.
  • Need probes / streaming-unlock detection? Use the CLI: go install github.com/Au1rxx/proxykit/cmd/proxykit@latest — those features need a sing-box subprocess, which can't run in a browser.

proxykit is MIT-licensed. Source on github.com/Au1rxx/proxykit. Sister project free-vpn-subscriptions publishes an hourly-verified public node feed (~150 alive nodes) that you can paste straight in here.

Heuristics & emit layers may change. Verify critical configs against the official sing-box / Clash docs before deploying.
